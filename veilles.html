Google Carbon

Carbon est un langage de programmation à usage général créé par Google pour remplacer le C++, présenté pour la première fois au public par Chandler Carruth lors de la conférence CppNorth en juillet 2022. Le langage est conçu pour corriger plusieurs défauts du C++ mais fournit par ailleurs un ensemble de fonctionnalités similaires. Les principaux objectifs du langage sont la lisibilité et "l'interopérabilité bidirectionnelle", pour éviter toute barrière linguistique comme pour Rust. Le langage sera conçu et développé sur GitHub. Le langage est développé en tant que projet libre et open source sous la licence Apache2
Code d'affichage "hello world" en Google Charbon

  package sample api;

  fn Main() -> i32 {

    Print("Hello, world!");

    return 0;

  }

Code d'affichage "hello world" en C++

  int main(){

    std::cout<< ("Hello, world!");

    return 0;

  }

Les outils pour s'informé sur cette veille sera la communauté discord officielle de carbon et leurs site github. On obtient directement les mises à jours du programme et du progrès effectué
